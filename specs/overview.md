Overview
=========================

プロジェクト
----------------------

開発環境
----------------------
- Node.js 25.5.0 (Volta管理)
- npm 11.8.0 (Volta管理)
- JavaScript (TypeScriptは使用しない)
- Astro 5.17.1
- Biome 2.3.13


デプロイ
----------------------
Github Actionsを利用して、Github Pageへリリースする。
